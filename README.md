# 卓著資料快速搜尋網頁

主要針對手機端設計的快速搜尋卓著所有歌曲相關資訊網站應用程式

[Demo](https://shinenic.github.io/zhuozhe-quick-search/)


<img src="demo1.gif" alt="demo1" width="220"/> <img src="demo2.gif" alt="demo1" width="220"/>




## 功能

* 搜尋
  * 支援模糊搜尋，可透過歌名、歌手、冊數、頁碼找尋
  * 點選歌名三下可連結至 Youtube 搜尋
  * 點選歌手可搜尋該歌手之所有曲目
  * 預設僅載入 1 至 104 冊資料
* 紀錄
  * 搜尋紀錄可於側邊選單找尋資料





## 使用技術

* 長列表效能優化
* Redux
* MiddleWare
* React-Router
